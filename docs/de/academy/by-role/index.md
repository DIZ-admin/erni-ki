# Szenarien nach Rolle

Wahlen Sie Ihre Rolle, um relevante KI-Nutzungsszenarien zu finden.

## Verfugbare Rollen

### [Entwickler](developers/index.md)

KI fur Software-Entwicklung, Debugging, Testing und Code-Qualitat.

- Code Review mit KI
- Code debuggen
- Unit Tests schreiben

### [Manager](managers/index.md)

KI fur Projektmanagement, Datenanalyse, Reporting und Team-Fuhrung.

- Projektberichte erstellen
- Metriken analysieren

### [Support & QA](support/index.md)

KI fur Customer Support, Troubleshooting, Knowledge Base und Qualitatssicherung.

- Benutzerprobleme diagnostizieren
- Knowledge Base Artikel erstellen

### [Allgemeine Benutzer](general-users/index.md)

Universelle Szenarien fur alle Mitarbeiter.

- Professionelle E-Mails schreiben
- Dokumente ubersetzen
- Prasentationen vorbereiten
