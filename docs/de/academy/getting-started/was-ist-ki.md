# Was ist KI und Large Language Models

## Was Sie lernen werden

- Was ist Künstliche Intelligenz (KI)
- Was sind Large Language Models (LLM)
- Wie moderne KI-Modelle funktionieren
- Was ist der Unterschied zwischen verschiedenen KI-Typen

## ⏱ Benötigte Zeit

15 Minuten

## Was ist Künstliche Intelligenz?

**Künstliche Intelligenz (KI)** sind Computersysteme, die Aufgaben ausführen
können, die normalerweise menschliche Intelligenz erfordern:

- Verstehen von Text und Sprache
- Problemlösung
- Entscheidungsfindung
- Lernen aus Erfahrung

### KI-Typen

1. **Schwache KI (Narrow AI)**

- Spezialisiert auf eine bestimmte Aufgabe
- Beispiele: Gesichtserkennung, Textübersetzung
- Das ist, was wir heute verwenden

2. **Starke KI (General AI)**

- Kann jede intellektuelle Aufgabe wie ein Mensch ausführen
- Existiert noch nicht
- Ist das Ziel zukünftiger Entwicklung

## Was sind Large Language Models (LLM)?

**Large Language Models (LLM)** sind ein KI-Typ, der sich auf Textverarbeitung
spezialisiert:

- **Trainiert auf riesigen Textmengen** aus dem Internet, Büchern, Artikeln
- **Verstehen Kontext** und Bedeutung des Geschriebenen
- **Generieren Text**, der natürlich wirkt
- **Multitasking**: können schreiben, übersetzen, Fragen beantworten, Code
  schreiben

### Wie funktioniert ein LLM?

1. **Training**

- Modell liest Milliarden von Texten
- Lernt das nächste Wort vorherzusagen
- Merkt sich Sprachmuster

2. **Inference** (Inferenz)

- Sie geben eine Anfrage (Prompt)
- Modell analysiert den Kontext
- Generiert die wahrscheinlichste Antwort

### Einfache Analogie

Stellen Sie sich ein LLM wie eine sehr belesene Person vor:

- Hat Millionen von Büchern, Artikeln, Dialogen gelesen
- Kann sich über jedes Thema unterhalten
- Hat aber kein echtes Verständnis oder Bewusstsein
- Kann einfach sehr gut vorhersagen, was als nächstes gesagt werden sollte

## Beliebte LLM-Modelle

### GPT-4 (OpenAI)

- Eines der mächtigsten Modelle
- Versteht Kontext ausgezeichnet
- Funktioniert gut mit Code
- **Verwenden Sie für:** komplexe Aufgaben, Analysen, Programmierung

### Claude (Anthropic)

- Fokus auf Sicherheit und Genauigkeit
- Folgt Anweisungen gut
- Weniger "Halluzinationen"
- **Verwenden Sie für:** präzise Antworten, sichere Aufgaben

### Llama (Meta)

- Open-Source-Modell
- Kann lokal ausgeführt werden
- Gute Leistung
- **Verwenden Sie für:** Basisaufgaben, Experimente

## Wichtige Einschränkungen von LLM

### Was LLM NICHT kann:

1. **Kein echtes Verständnis**

- Sagt nur Text voraus
- Kein echtes Bewusstsein oder Denken

2. **Kann "halluzinieren"**

- Erfindet manchmal Fakten
- Kann selbstbewusst klingen, aber falsch liegen
- **Überprüfen Sie immer wichtige Informationen!**

3. **Kennt keine aktuellen Nachrichten**

- Wissen ist auf Trainingsdatum begrenzt
- Hat keinen Internetzugang (wenn nicht RAG angeschlossen)
- Kennt keine Ereignisse nach dem Training

4. **Ersetzt nicht menschliches Urteilsvermögen**

- Werkzeug zur Unterstützung, kein Ersatz für Menschen
- Kritisches Denken bleibt wichtig
- Endgültige Entscheidungen liegen bei Ihnen

## Wie hilft KI bei der Arbeit?

### Aufgaben, bei denen KI ein ausgezeichneter Assistent ist:

**Texte schreiben**

- Entwürfe von E-Mails, Berichten
- Verbesserung von Formulierungen
- Übersetzung in andere Sprachen

**Mit Code arbeiten**

- Hilfe beim Schreiben von Code
- Erklärung komplexer Fragmente
- Fehlersuche

**Analyse und Zusammenfassung**

- Zusammenfassung langer Dokumente
- Extraktion von Hauptpunkten
- Vergleich von Optionen

**Lernen**

- Erklärung komplexer Konzepte
- Antworten auf Fragen
- Praktische Beispiele

### Aufgaben, bei denen Vorsicht geboten ist:

**Faktische Informationen**

- Überprüfen Sie Fakten aus anderen Quellen
- Vertrauen Sie Statistiken nicht blind
- Überprüfen Sie Daten und Zahlen

**Kritische Entscheidungen**

- Treffen Sie keine wichtigen Entscheidungen nur aufgrund von KI
- Verwenden Sie als eine der Informationsquellen
- Wenden Sie professionelles Urteilsvermögen an

**Vertrauliche Daten**

- Geben Sie keine Firmengeheimnisse ein
- Teilen Sie keine persönlichen Kundeninformationen
- Befolgen Sie die Sicherheitsrichtlinien

## Einfacher Vergleich: KI vs Mensch

| Aspekt                    | LLM          | Mensch        |
| ------------------------- | ------------ | ------------- |
| Geschwindigkeit           | Sehr schnell | Langsamer     |
| Verarbeitungsvolumen      | Riesig       | Begrenzt      |
| Kreativität               | Gut          | Ausgezeichnet |
| Kontextverständnis        | Grundlegend  | Tiefgehend    |
| Kritisches Denken         | Nein         | Ja            |
| Emotionale Intelligenz    | Nein         | Ja            |
| Ethisches Urteilsvermögen | Begrenzt     | Entwickelt    |

## Schlüsselbegriffe

**KI (Künstliche Intelligenz)** — Computersysteme, die Aufgaben ausführen, die
Intelligenz erfordern

**LLM (Large Language Model)** — Großes Sprachmodell, ein KI-Typ für
Textverarbeitung

**Prompt** — Anfrage oder Anweisung, die Sie der KI geben

**Halluzination** — Wenn KI nicht existierende Fakten erfindet

**Kontext** — Informationen, die KI verwendet, um Ihre Anfrage zu verstehen
