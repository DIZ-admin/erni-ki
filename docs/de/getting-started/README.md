## Voraussetzungen

- Ubuntu 20.04+ oder Debian 11+
- Docker 20.10+
- 8GB RAM Minimum (32GB empfohlen)
- 50GB Speicher Minimum (200GB+ empfohlen)
- NVIDIA GPU optional (RTX 4060+ empfohlen)
