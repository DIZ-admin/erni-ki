# Промптинг 101

## Что такое промпт

Промпт — это чёткая инструкция модели: кто вы, что нужно сделать, какой контекст
учитывать и в каком формате вернуть ответ. Чем конкретнее запрос, тем быстрее вы
получите понятный результат.

## Базовый шаблон

**РОЛЬ → ЗАДАЧА → КОНТЕКСТ → ФОРМАТ**

- Роль: от чьего лица и с какой целью отвечает ассистент.
- Задача: что нужно сделать.
- Контекст: исходные данные, ограничения, получатель.
- Формат: тип ответа (список, письмо, таблица, маркдаун).

## Примеры

### Письмо клиенту

- Роль: «Ты аккаунт-менеджер ERNI».
- Задача: «Подготовь письмо о продлении поддержки».
- Контекст: «Клиенту важны SLA и прозрачный план».
- Формат: «3 абзаца, дружелюбный тон».  
  **Пример промпта:**

```
Ты аккаунт-менеджер ERNI. Подготовь письмо клиенту о продлении поддержки. Клиенту важны SLA и прозрачный план. Сформируй 3 абзаца: приветствие, предложение продления с выгодами, шаги по запуску.
```

### Резюме митинга

- Роль: «Ты фасилитатор встречи».
- Задача: «Сделай краткое резюме».
- Контекст: «Транскрипт ниже, важны решения и владельцы».
- Формат: «Маркированный список: цели, решения, следующие шаги». **Пример
  промпта:**

```
Ты фасилитатор встречи. Сделай резюме по транскрипту ниже. Нужны цели, принятые решения и владельцы следующих шагов. Дай ответ маркированным списком.
<вставьте текст встречи>
```

### Улучшение текста документа

- Роль: «Ты редактор».
- Задача: «Улучшить ясность и структуру».
- Контекст: «Исходный текст приведён ниже, аудитория — инженеры».
- Формат: «Верни улучшенную версию + 3 ключевых правки». **Пример промпта:**

```
Ты редактор. Улучшай ясность и структуру текста для инженерной аудитории. Верни улучшенную версию и перечисли 3 ключевые правки. Текст: <вставьте фрагмент>.
```

## Минимальный уровень

Эти примеры покрывают базовые сценарии. Если нужен другой результат, уточняйте
роль, ожидаемый формат и ограничения прямо в промпте.
