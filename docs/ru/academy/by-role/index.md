## Как использовать сценарии

### Шаг 1: Найдите релевантный сценарий

Выберите сценарий из вашей профессиональной области или используйте поиск.

### Шаг 2: Изучите пример

Каждый сценарий содержит:

- Описание задачи
- Шаблон промпта
- Практические примеры
- Best practices

### Шаг 3: Адаптируйте под свою задачу

Используйте шаблоны как основу, модифицируйте под ваши нужды.

### Шаг 4: Сохраняйте успешные промпты

Создайте личную библиотеку эффективных промптов для повторного использования.

## Цели обучения по ролям

### Developers

**Базовый уровень (3 сценария, ~45 мин):**

- Использовать ИИ для code review
- Диагностировать и фиксить баги
- Генерировать unit tests

**Результат:** Ускорение development процесса на 30-40%

---

### Managers

**Базовый уровень (2 сценария, ~25 мин):**

- Создавать project status reports
- Анализировать metrics и data

**Результат:** Экономия 2-3 часов в неделю на reporting

---

### Support

**Базовый уровень (2 сценария, ~20 мин):**

- Эффективно диагностировать проблемы
- Создавать качественные KB статьи

**Результат:** Снижение ticket resolution time на 25%

---

### All Users

**Базовый уровень (3 сценария, ~35 мин):**

- Писать профессиональные emails
- Переводить документы
- Готовить презентации

**Результат:** Улучшение качества communication на 40%
