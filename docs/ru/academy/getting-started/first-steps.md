# Первые шаги в Open WebUI

## Что вы узнаете

- Как войти в Open WebUI
- Как устроен интерфейс
- Как выбрать модель ИИ
- Как задать ваш первый запрос
- Основные элементы управления

## ⏱ Требуемое время

20 минут

## Предварительные требования

- Прочитали [Что такое ИИ](what-is-ai.md)
- Имеете доступ к корпоративной сети ERNI
- Есть учётные данные для входа

## Шаг 1: Вход в систему

### Доступ к Open WebUI

1. **Откройте браузер** (рекомендуем Chrome, Firefox или Edge)

2. **Перейдите по адресу:**

- Внутренняя сеть: `https://ki.erni-gruppe.ch`
- VPN: используйте предоставленный адрес

3. **Войдите через SSO:**

- Нажмите кнопку "Login with SSO"
- Используйте ваши корпоративные учётные данные
- Подтвердите двухфакторную аутентификацию (если требуется)

### Проверка доступа

После входа вы должны увидеть:

- Главную страницу с полем ввода
- Список доступных моделей в верхней части
- Боковую панель с меню

  **Если доступ не работает:**

1. Проверьте [статус системы](https://status.erni-ki.ch)
2. Убедитесь, что вы в корпоративной сети
3. Создайте тикет в поддержку

## Шаг 2: Знакомство с интерфейсом

### Основные элементы

```

 [≡] Open WebUI [Model ] [@] Profile



 Ваши разговоры появятся здесь






 Здесь отображаются ответы ИИ




 Введите ваше сообщение...


```

### Боковое меню (слева):

- **Home** — главная страница
- **Chats** — история ваших диалогов
- **Workspaces** — рабочие пространства с шаблонами
- **Settings** — настройки профиля

### Верхняя панель:

- **Model Selector** — выбор модели ИИ
- **Profile Menu** — настройки профиля, выход

### Основная область:

- **Область диалога** — здесь отображаются ваши сообщения и ответы ИИ
- **Поле ввода** — сюда вы пишете запросы
- **Кнопки:** прикрепить файл, голосовой ввод

## Шаг 3: Выбор модели

### Как выбрать модель?

1. **Нажмите на селектор модели** в верхней части экрана
2. **Просмотрите доступные модели** с описаниями
3. **Выберите подходящую** для вашей задачи

### Какую модель выбрать?

#### Для начинающих:

- **GPT-4o** — универсальная, хорошо справляется с большинством задач
- Рекомендуем начать с неё!

#### Для конкретных задач:

**Написание текстов:**

- GPT-4o — отлично для писем, отчётов
- Claude — более точные и безопасные ответы

**Программирование:**

- GPT-4o — лучше для кода
- Claude — хорошо объясняет код

**Работа с документами (RAG):**

- Модели с пометкой "RAG-enabled"
- Могут искать в загруженных документах

**Локальная модель:**

- Llama — работает на сервере ERNI
- Быстрая, но менее мощная

### Совет новичкам

> Не уверены? Начните с **GPT-4o** — это универсальный выбор для 90% задач.

## Шаг 4: Ваш первый запрос

### Базовая структура запроса

**Хороший запрос включает:**

1. Контекст (кто вы, что нужно)
2. Задачу (что конкретно сделать)
3. Формат (как должен выглядеть результат)

### Пример первого запроса

```
Привет! Я новый пользователь Open WebUI.
Можешь кратко объяснить, как ты можешь мне помочь в работе?
Дай 3-4 примера задач, с которыми ты справляешься.
```

### Отправка запроса

1. **Введите текст** в поле ввода
2. **Нажмите Enter** или кнопку отправки
3. **Дождитесь ответа** (обычно 2-10 секунд)

### Что делать с ответом

- **Прочитайте внимательно** — ИИ часто даёт подробные ответы
- **Задайте уточняющий вопрос** — если что-то непонятно
- **Скопируйте полезное** — используйте кнопку копирования
- **Оцените ответ** — или помогает улучшать систему

## Шаг 5: Диалог с ИИ

### ИИ помнит контекст беседы

```
Вы: Расскажи про Python

ИИ: Python — это язык программирования...

Вы: А какие у него преимущества?

ИИ: У Python есть несколько преимуществ... [помнит, что речь про Python]
```

### Если нужно начать новую тему

- **Нажмите "+ New Chat"** в верхнем левом углу
- Или просто скажите "Давай сменим тему"

### Длина разговора

- ИИ помнит историю текущего чата
- Но есть лимит на длину контекста (~4000-8000 слов)
- При длинном диалоге начинайте новый чат

## Шаг 6: Полезные функции

### Прикрепление файлов

1. Нажмите значок в поле ввода
2. Выберите файл (поддерживаются: PDF, DOCX, TXT, изображения)
3. ИИ прочитает и проанализирует содержимое

**Пример использования:**

```
Прикрепите PDF-документ

Вы: Сделай краткое резюме этого документа в 5 пунктах
```

### Голосовой ввод

1. Нажмите значок
2. Разрешите доступ к микрофону
3. Говорите ваш запрос
4. Нажмите снова для завершения записи

### История чатов

- Все ваши разговоры сохраняются
- Доступ через боковое меню → Chats
- Можете продолжить любой старый диалог

### Рабочие пространства

- Готовые шаблоны для типовых задач
- Преднастроенные промпты
- Специализированные модели

## Частые ошибки новичков

### Слишком короткие запросы

```
Плохо: "Напиши письмо"
Хорошо: "Напиши короткое официальное письмо клиенту с извинениями
за задержку поставки на 2 дня"
```

### Отсутствие контекста

```
Плохо: "Исправь это"
Хорошо: "Вот код на Python, который выдаёт ошибку. Можешь найти проблему
и предложить исправление?"
```

### Ожидание идеального первого ответа

```
Помни: ИИ — это диалог! Если ответ не идеален, уточните:
"Можешь сделать это короче?"
"Объясни более простыми словами"
"Добавь больше деталей про X"
```

## Советы для начинающих

1. **Экспериментируйте!** Попробуйте разные типы запросов
2. **Уточняйте!** Если ответ не подходит, просто попросите изменить
3. **Используйте примеры** В запросе покажите, что вы хотите
4. **Будьте вежливы** Это не обязательно, но делает диалог приятнее
5. **Не бойтесь ошибок** Худшее что может случиться — неудачный ответ, просто
   попробуйте снова
