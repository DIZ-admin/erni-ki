# Примеры интеграций ERNI-KI

Набор готовых примеров для работы с вебхуками и конфигурацией сервисов.

- `webhook-client-python.py` — CLI-клиент на Python с HMAC-подписью запросов.
- `webhook-examples.sh` — Bash-функции для отправки вебхуков через curl/openssl.
- `docker-compose-override.yml` — пример override-файла для локального запуска.
- `litellm-config.yaml` — пример конфигурации LiteLLM.

Смотрите также `docs/en/examples/index.md` для англоязычного описания.
