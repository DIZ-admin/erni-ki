# Что такое ИИ и Large Language Models

## Что вы узнаете

- Что такое искусственный интеллект (ИИ)
- Что такое Large Language Models (LLM)
- Как работают современные ИИ-модели
- В чём разница между разными типами ИИ

## ⏱ Требуемое время

15 минут

## Что такое Искусственный Интеллект?

**Искусственный интеллект (ИИ)** — это компьютерные системы, которые могут
выполнять задачи, обычно требующие человеческого интеллекта:

- Понимание текста и языка
- Решение проблем
- Принятие решений
- Обучение на основе опыта

### Типы ИИ

1. **Узкий ИИ (Narrow AI)**

- Специализируется на одной конкретной задаче
- Примеры: распознавание лиц, перевод текста
- Это то, что мы используем сегодня

2. **Общий ИИ (General AI)**

- Может выполнять любые интеллектуальные задачи как человек
- Пока не существует
- Это цель будущего развития

## Что такое Large Language Models (LLM)?

**Large Language Models (LLM)** — это тип ИИ, специализирующийся на работе с
текстом:

- **Обучены на огромных объёмах текста** из интернета, книг, статей
- **Понимают контекст** и смысл написанного
- **Генерируют текст** который выглядит естественно
- **Многозадачны**: могут писать, переводить, отвечать на вопросы, писать код

### Как работает LLM?

1. **Обучение** (Training)

- Модель читает миллиарды текстов
- Учится предсказывать следующее слово
- Запоминает паттерны языка

2. **Инференс** (Inference)

- Вы даёте запрос (prompt)
- Модель анализирует контекст
- Генерирует наиболее вероятный ответ

### Простая аналогия

Представьте LLM как очень начитанного человека:

- Прочитал миллионы книг, статей, диалогов
- Может поддержать беседу на любую тему
- Но не имеет настоящего понимания или сознания
- Просто очень хорошо предсказывает, что нужно сказать дальше

## Популярные LLM модели

### GPT-4 (OpenAI)

- Одна из самых мощных моделей
- Отлично понимает контекст
- Хорошо работает с кодом
- **Используйте для:** сложных задач, анализа, программирования

### Claude (Anthropic)

- Фокус на безопасности и точности
- Хорошо следует инструкциям
- Меньше "галлюцинаций"
- **Используйте для:** точных ответов, безопасных задач

### Llama (Meta)

- Open source модель
- Можно запускать локально
- Хорошая производительность
- **Используйте для:** базовых задач, эксперименты

## Важные ограничения LLM

### Что LLM НЕ может:

1. **Не имеет реального понимания**

- Просто предсказывает текст
- Нет настоящего сознания или мышления

2. **Может "галлюцинировать"**

- Иногда придумывает факты
- Может звучать уверенно, но ошибаться
- **Всегда проверяйте важную информацию!**

3. **Не знает последних новостей**

- Знания ограничены датой обучения
- Не имеет доступа к интернету (если не подключен RAG)
- Не знает о событиях после обучения

4. **Не заменяет человеческое суждение**

- Инструмент для помощи, не замена человека
- Критическое мышление остаётся важным
- Финальные решения — за вами

## Как ИИ помогает в работе?

### Задачи, где ИИ отличный помощник:

**Написание текстов**

- Черновики писем, отчётов
- Улучшение формулировок
- Перевод на другие языки

  **Работа с кодом**

- Помощь в написании кода
- Объяснение сложных фрагментов
- Поиск ошибок

  **Анализ и обобщение**

- Суммаризация длинных документов
- Извлечение ключевых пунктов
- Сравнение опций

  **Обучение**

- Объяснение сложных концепций
- Ответы на вопросы
- Практические примеры

### Задачи, где нужна осторожность:

**Фактическая информация**

- Проверяйте факты из других источников
- Не доверяйте слепо статистике
- Перепроверяйте даты и числа

  **Критические решения**

- Не принимайте важные решения только на основе ИИ
- Используйте как один из источников информации
- Применяйте профессиональное суждение

  **Конфиденциальные данные**

- Не вводите секретные данные компании
- Не делитесь персональной информацией клиентов
- Следуйте политике безопасности

## Простое сравнение: ИИ vs Человек

| Аспект                  | LLM          | Человек      |
| ----------------------- | ------------ | ------------ |
| Скорость                | Очень быстро | Медленнее    |
| Объём обработки         | Огромный     | Ограниченный |
| Креативность            | Хорошая      | Отличная     |
| Понимание контекста     | Базовое      | Глубокое     |
| Критическое мышление    | Нет          | Да           |
| Эмоциональный интеллект | Нет          | Да           |
| Этические суждения      | Ограниченные | Развитые     |

## Ключевые термины

**ИИ (Искусственный Интеллект)** — компьютерные системы, выполняющие задачи,
требующие интеллекта

**LLM (Large Language Model)** — большая языковая модель, тип ИИ для работы с
текстом

**Prompt** — запрос или инструкция, которую вы даёте ИИ

**Галлюцинация** — когда ИИ придумывает несуществующие факты

**Контекст** — информация, которую ИИ использует для понимания вашего запроса
