## Как эффективно учиться

### 1. Теория + Практика

```
Не просто читайте — пробуйте!

Для каждой техники:
1. Прочитайте объяснение
2. Посмотрите пример
3. Попробуйте сами в Open WebUI
4. Экспериментируйте с вариациями
```

### 2. Ведите заметки

```
Создайте свой "промпт-журнал":
- Удачные промпты для повторного использования
- Неудачные попытки и их причины
- Insights и открытия
- Шаблоны для типовых задач
```

### 3. Постепенное улучшение

```
Не пытайтесь освоить всё сразу:
- Начните с основ промптинга
- Практикуйте 1-2 дня
- Добавляйте новые техники постепенно
- Закрепляйте успехи
```

## Практические упражнения

### Базовые упражнения (после Prompting Fundamentals)

**Упражнение 1: Улучшение промпта**

```
Возьмите простой промпт: "Расскажи про Docker"

Улучшите его используя структуру:
- Роль
- Задача
- Контекст
- Формат

Сравните результаты!
```

**Упражнение 2: Итеративное уточнение**

```
Начните с общего запроса о Python
Уточняйте 3-4 раза пока не получите идеальный ответ
Обратите внимание как ИИ адаптируется
```

**Упражнение 3: Форматирование вывода**

```
Попросите ИИ дать информацию в разных форматах:
- Таблица
- Bullet points
- Code
- JSON
```

### Продвинутые упражнения (после всех материалов)

**Упражнение 4: Chain of Thought**

```
Дайте ИИ сложную задачу
Попросите показать рассуждения step-by-step
Проанализируйте логику
```

**Упражнение 5: Few-shot Learning**

```
Дайте 2-3 примера желаемого формата
Попросите создать новый в том же стиле
Оцените consistency
```

**Упражнение 6: RAG практика**

```
Загрузите PDF документ
Задайте 5 вопросов разной сложности
Проверьте точность ответов
```
