## Prerequisites

- Ubuntu 20.04+ or Debian 11+
- Docker 20.10+
- 8GB RAM minimum (32GB recommended)
- 50GB storage minimum (200GB+ recommended)
- NVIDIA GPU optional (RTX 4060+ recommended)
